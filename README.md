# 📚 Library Book Manager

A modern **React-based Library Book Manager** that allows users to efficiently manage books, track availability, and organize their personal or institutional library.

🔗 **DEPLOYED AT:** [https://manibook-library.netlify.app/](https://manibook-library.netlify.app/)

---

## 🚀 Features

* 📖 Add, edit, and delete books
* 🔍 Search and filter books
* 📚 View all books in a clean UI
* ⚡ Fast and responsive React interface
* 🌐 Deployed on Netlify

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** CSS / Tailwind (update if needed)
* **Deployment:** Netlify

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone <your-repo-link>

# Navigate to project directory
cd book-library

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 🏗️ Build & Deployment

To create a production build:

```bash
npm run build
```

Deploy the `dist` folder to Netlify.
